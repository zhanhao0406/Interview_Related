### 定义方式
1. 数据类型 数组名[数组长度]；
2. 数据类型 数组名[数组长度] = {val 1, val 2, ...}；
3. 数据类型 数组名[ ] = {val 1, val 2, ...}；

下标：从 0 开始；

---
### 一维数组 
arr;
1. 可以统计整个数组在内存中的长度： `sizeof(arr)`
2. 第一个的长度： `sizeof(arr[0])`;
3. 可以获取数组在内存中的首地址 `cout << arr`
4. 第一个元素的地址： `(int) &arr[0]` ;

### <mark>冒泡排序</mark>
由小从大排。 4 2 8 0 5 7 1 3 9；  
提示：  
排序总轮数 = 元素个数 - 1；   
每轮对比次数 = 元素个数 - 排序轮数 - 1；  
<summary> 具体代码 </summary>
<details>
  
```cpp
#include <iostream>

using namespace std;
int main (){

    int arr[] = {4,2,8,0,5,7,1,3,9};
    int len = sizeof(arr)/sizeof(arr[0]);
    //cout << len << endl;

    cout << "排序前： " << endl;
    for (int i =0; i < len ; i++){
        
        cout << arr[i] << " " ;
    }

    cout << "排序后： " << endl;
    for (int i=0; i < len-1 ; i++){
        for (int j=0; j < len -i-1;j++){
            if (arr[j]>arr[j+1]){
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
     for (int i =0; i < len ; i++){
        
        cout << arr[i] << " " ;
    }


    return 0;
}

```  
</details>

### 二维数组
1. 数据类型 数组名[行数][列数];
2. 数据类型 数组名[行数][列数] = {{val 1, val 2},{val 3, val 4}};
3. 数据类型 数组名[行数][列数] = {val 1, val 2,val 3, val 4};
4. 数据类型 数组名[ ][列数] = {val 1, val 2,val 3, val 4};

下标：从 0 开始；
